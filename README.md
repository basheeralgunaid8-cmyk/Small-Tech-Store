Laptop Store Management System

The Laptop Store System is a Windows Forms desktop application developed using C#.
It simulates a real online laptop store where users can browse different laptop brands and models, view detailed specifications, customize configurations, and manage a shopping cart.

The system allows users to:

Select laptop brands and models

View detailed specifications (processor, RAM, screen, battery, weight, etc.)

Choose memory and color options

Add products to a cart

Adjust quantities

Automatically calculate subtotal, tax, delivery, and total price

Clear cart or start a new order

Switch application view modes (Dark Mode, etc.)

Access help, about, and contact information

The application focuses on:

User Interface design

Event-driven programming

Control management (ListView, ComboBox, MenuStrip, etc.)

Logical price calculations

Structured project organization

This project demonstrates understanding of:

Windows Forms controls

Application structure

Business logic implementation

UI/UX basic principles
